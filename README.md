# USD to INR Prediction using Machine Learning

## Overview

This project predicts the next day's USD to INR exchange rate using a Random Forest Regression model.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- yfinance

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python Predict.py
```

## Machine Learning Model

Random Forest Regressor

## Data Source

Yahoo Finance

Ticker:

```
INR=X
```

## Output

- Predicted USD/INR value
- Historical graph
